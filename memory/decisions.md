| 2026-07-12 | Use timestep indices `n` and `n+1` in the numerical-integration update equations, with `i` as the pole index. | This is standard numerical-method notation, avoids italic word-like subscripts such as `next`, makes simultaneous updates precise, and scales naturally to multiple poles. | Use upright descriptive subscripts such as `x_{\mathrm{next}}`; leave state values unindexed. |
