# Agent Instructions

This is the repository for the SharpNEAT website. It is a static site: public web content lives under `public/`, archived pages live under `archive/`, and source artwork lives under `artwork/`.

Use the Lode Working process in [lode-working.md](lode-working.md). At the start of a task, read [memory/summary.md](memory/summary.md) and [memory/map.md](memory/map.md), then read only the files relevant to the requested work.

Preserve the existing static HTML/CSS structure unless the user asks for a broader redesign or migration. Do not modify or revert unrelated dirty worktree changes.
